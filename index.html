Destructuración de Objetos y Arrays. Funciones flechas

La destructuración es una característica poderosa en JavaScript que permite extraer valores de objetos y arreglos de una manera más concisa. Aquí veremos ejemplos de cómo usarla:

Destructuración de Objetos

La destructuración de objetos nos permite extraer propiedades de un objeto y asignarlas a variables:


    const persona = { nombre: 'John', edad: 30, ciudad: 'Nueva York' };
    const { nombre, edad, ...rest } = persona;

Destructuración de Arreglos

La destructuración de arreglos nos permite extraer elementos de un arreglo y asignarlos a variables:


    const colores = ['rojo', 'verde', 'azul'];
    const [color1, color2, ...rest] = colores;

Métodos de Arreglos

En JavaScript, existen varios métodos útiles para manipular arreglos, como reduce, map, filter, forEach, find y el operador de propagación (spread operator):

  • reduce: Utilizado para reducir un arreglo a un valor único.
  • map: Crea un nuevo arreglo aplicando una función a cada elemento del arreglo original.
  • filter: Crea un nuevo arreglo con elementos que cumplen una condición específica.
  • forEach: Ejecuta una función en cada elemento del arreglo.
  • find: Encuentra el primer elemento en un arreglo que cumple una condición.
  • Spread Operator: Permite combinar múltiples arreglos o clonar objetos.
  • Rest Operator: Captura elementos restantes en un arreglo o propiedades en un objeto.

Funciones Flecha en JavaScript

Las funciones flecha son una característica introducida en ECMAScript 6 (ES6) que proporcionan una forma más concisa de definir funciones en JavaScript en comparación con las funciones regulares.

Ejemplo de función flecha:


    const sumar = (a, b) => a + b;
    console.log(sumar(2, 3)); // 5
